# 📊 Data Usage Policy (Multimodal Clinical AI)

## 🎯 Objetivo

Definir regras de uso de dados clínicos multimodais (imagem, vídeo e telemetria) para pesquisa e desenvolvimento de IA.

---

## 🧠 Tipos de dados

### 🖼️ Imagem médica
- Raio-X
- CT
- MRI
- Ultrassom

### 🎥 Vídeo clínico
- Cirurgias
- Monitoramento de UTI
- Procedimentos invasivos

### 📈 Telemetria
- ECG
- SpO2
- Pressão arterial
- Frequência cardíaca

---

## ⚖️ Regras de uso

- Dados só podem ser usados se anonimizados ou pseudonimizados
- Uso exclusivamente para pesquisa ou validação clínica aprovada
- Proibido uso fora do escopo definido no projeto

---

## 🚫 Proibições

- Reidentificação de pacientes
- Uso comercial sem aprovação ética
- Treino com dados sem rastreabilidade completa

# 🧪 Experiments

> Espaço dedicado à exploração, validação e evolução de ideias ao longo do doutorado.

---

## 📌 Visão Geral

A pasta `experiments/` reúne experimentos independentes, cada um representando uma hipótese, abordagem ou variação metodológica dentro da linha de pesquisa.

Cada experimento é estruturado para ser:

- Reprodutível  
- Modular  
- Interpretável  
- Evolutivo (capaz de gerar novos experimentos)  

---

## 🧠 Filosofia

Os experimentos não são apenas testes técnicos.

Eles representam:

- Perguntas de pesquisa  
- Tentativas de resposta  
- Evidências (quantitativas e qualitativas)  
- Limitações e próximos passos  

---

## 📁 Estrutura

Cada experimento segue um padrão mínimo:

```bash
experiment-name/
├── README.md
├── experiment.py
├── configs/
├── models/
├── training/
├── evaluation/
├── visualization/
├── results/
└── notebooks/

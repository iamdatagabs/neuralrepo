## Código-Fonte (src/)

Este diretório contém a implementação central do framework experimental para treinamento, avaliação e reprodutibilidade de redes neurais.

---

## 🧠 Visão Geral

O módulo src/ é responsável por:

Definições de modelos
Pipelines de treinamento
Métricas de avaliação
Configuração de experimentos
Funções utilitárias

Ele é projetado para ser modular, reutilizável e reprodutível, suportando múltiplas arquiteturas e diferentes configurações experimentais.

---

## 📂 Estrutura
```
src/
├── data/          # Manipulação de datasets e pré-processamento
├── models/        # Arquiteturas de redes neurais
├── training/      # Lógica e pipelines de treinamento
├── evaluation/    # Métricas e scripts de avaliação
├── experiments/   # Arquivos de configuração YAML
├── utils/         # Funções utilitárias
└── __init__.py
```
